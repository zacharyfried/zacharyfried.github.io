# zacharyfried.github.io

Personal site: a single hand-written HTML page (no framework, no build step),
served directly by GitHub Pages.

- `index.html` + `style.css` - the whole site
- `files/` - resume PDFs (stable paths; `Zachary_Fried_Public_Resume.pdf` is current)
- `images/` - email-address image and personal photos/figures
- `cv/`, `resume/`, `about/`, `portfolio/` - redirect stubs preserving old Jekyll URLs
- `.nojekyll` - disables Jekyll processing on GitHub Pages

To update: edit `index.html`, keep copy consistent with the current public resume,
and push to `master`. The previous academicpages/Jekyll site lives in git history
(pre-2026 rebuild).
